Abstract

We tackle the challenging task of All-Weather Image Restoration (AWIR) under adverse weather conditions. Prior works do not utilize degradation-specific visual contextual guidance during training.Consequently, the performance of existing methods is limited by the degradation cues that are learnt from individual training samples. We propose All-Weather Image Restoration using Visual In-Context Learning (AWRaCLe), a novel approach for AWIR which leverages dergadation-specific visual context information as a prior to enhance the restoration performance. Our approach incorporates Degradation Context Extraction (DCE) and Context Fusion (CF) to seamlessly integrate degradation-specific features from the context into an image restoration network. The proposed DCE and CF modules leverage robust CLIP features and incorporate attention mechanisms to adeptly learn and fuse contextual information. AWRaCLe achieves state-of-the-art performance on standard all-weather image restoration benchmarks.

Proposed Approach

AWRaCLe incorporates relevant degradation-specific information from a context pair in order to facilitate the iamge restoration process. Initially, CLIP features are extracted from the context pair and are subsequently fed to DCE blocks present at different decoder levels of an image restoration network. CF blocks then fuse the degradation information obtained from the DCE blocks with decoder features from the query image which needs to be restored. Finally, the restored image is generated.
